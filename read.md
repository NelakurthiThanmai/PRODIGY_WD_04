# Task 4 - To-Do List Web Application

## Overview

The To-Do List Web Application is a simple and interactive task management tool developed using HTML, CSS, and JavaScript. It helps users organize their daily activities by allowing them to create, update, complete, and remove tasks efficiently.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* User-friendly interface
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## Learning Outcomes

Through this project, I gained practical experience in:

* DOM Manipulation
* Event Handling
* Dynamic Content Updates
* Front-End Web Development

## Conclusion

This project demonstrates the implementation of core JavaScript concepts to build an interactive and productive web application for managing daily tasks.
